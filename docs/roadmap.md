# Roadmap

## Current Direction

Learn Portuguese / Nenoling is a beginner-friendly Portuguese learning app for Danish-speaking users.

Core experience:

- One phrase at a time
- Danish translation
- Grammar note
- Small glossary
- Previous/Next navigation
- Portuguese pronunciation through Android TextToSpeech

## Learn Portuguese 2

Planned level 2 work:

- Add `ser`, `estar`, and `ter` practice.
- Add more everyday dialogs.
- Add numbers from 20 to 100.
- Add shopping, supermarket, cafe, restaurant, and transport situations.
- Keep content easy to expand through JSON.

## Later Ideas

- Learn Portuguese 3-9
- Nenoling branding
- More quizzes and points
- Progress tracking
- Web version
- Play Store release
- More language pairs

## Release Readiness

Before public release:

- Verify the Android build.
- Verify Portuguese TTS behavior.
- Review language content.
- Confirm privacy and security docs match the actual app behavior.
