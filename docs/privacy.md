# Privacy

Learn Portuguese / Nenoling should be a simple offline-first learning app.

## Data Collection

The current project direction does not require collecting personal data.

The app should not collect:

- Name
- Email address
- Location
- Contacts
- Payment information
- Account credentials
- Learning history tied to a user identity

## Local Behavior

Lesson content should be stored locally in the app. Pronunciation uses Android TextToSpeech and may depend on voice data installed on the user's device.

## Network Use

The app should work offline except for any Android system behavior related to TextToSpeech voice availability or installation.

If future versions add accounts, analytics, cloud sync, AI services, or other network features, this document must be updated before release.

## Children And Beginners

The app should avoid collecting personal information from users. Content should remain suitable for beginner language learning.
